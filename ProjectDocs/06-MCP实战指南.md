# MCP 实战指南要点

## 6.1 对程序员
- **SQL 查询**：自然语言转 SQL → 数据库查询。
- **开发部署**：自然语言触发 GitLab/Jenkins/Slack。 
- **Manus 智能体**：典型多工具链场景。

## 6.2 对大众用户
- **旅游规划**：天气+交通+地图 API 联动。
- **联网搜索**：自定义搜索引擎。
- **业绩查询**：CRM + 财务系统 + 邮件。

## 6.3 热门 MCP Servers
- **filesystem**：本地文件操作。
- **mysqldb-mcp-server**：数据库交互。
- **amap-maps**：高德地图服务。
